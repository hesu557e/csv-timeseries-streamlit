# CSV Time-Series Streamlit Visualizer

A lightweight Streamlit web app for quick visualization and basic analysis of time-series CSV files.

## Features

- Drag & drop CSV upload
- Select X and Y columns
- Optional removal of first row
- Optional row range selection
- Interactive visualization
- Basic statistics (Max / Min / Mean)
